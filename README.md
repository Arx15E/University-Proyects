# University-Proyects
Proyectos de la universidad Inteligencia de datos :

Identifica los principios de la lógica borrosa en el modelamiento de variables aleatorias como variables lingüísticas

Integra información cualitativa proveniente de expertos como soporte a la toma de decisiones de las organizaciones en contextos bajo incertidumbre

Comprende los conceptos de la teoría de la decisión en la creación de escenarios prospectivos que permitan comprender a futuro el entorno
